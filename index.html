YouTubeThumbnail Grabber

This is a crude utility I've built to easily download YouTube video thumbnails. This is meant for my own purposes, but you're welcome to use it yourself.

YouTube ID

Enter the video's YouTube Id in the text box below. You can easily find it in the video's URL (https://www.youtube.com/watch?v=[ID HERE]&t=XXXX or https://youtu.be/[ID HERE]?t=XXXX)
